##  О проекте

Данный проект представляет собой полный пайплайн предобработки телеметрических данных нефтяных скважин с использованием библиотеки **NumPy**. 

В работе реализованы:
- ✅ Векторизованная обработка больших данных (~2 млн записей)
- ✅ Групповая нормализация и очистка выбросов
- ✅ Анализ временных рядов (скользящие окна, лаги, производные)
- ✅ Feature Engineering с учётом предметной области
- ✅ Оптимизация памяти через строгую типизацию

---

## 🎓 Цель и задачи

### Цель
Освоить методы высокопроизводительной обработки данных в Python с применением векторизованных вычислений NumPy на примере реальных телеметрических данных нефтегазовой отрасли.

### Основные задачи
1. Загрузка и первичный аудит данных
2. Векторизованная фильтрация аномалий
3. Групповая Z-score нормализация
4. Сглаживание временных рядов (O(N) алгоритмы)
5. Инженерия признаков (Feature Engineering)
6. Условная агрегация по группам
7. Анализ временных сдвигов (лаги)
8. Робастная замена выбросов (IQR-метод)
9. Логическая верификация данных
10. Частотный анализ и сжатие категорий
